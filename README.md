# Blood-Bank-Management-System
A Django-based Full Stack web application for managing blood donations, requests, and inventory. Built with Python, Django, MySQL, Bootstrap, and JavaScript.# 🩸 Blood Bank Management System

A web-based application built with **Python Django** to manage blood donations, donors, and recipients.  
This project ensures efficient tracking of blood availability and simplifies the process for hospitals and donors.

---

## 🚀 Features
- User authentication (Admin, Donor, Recipient)
- Donor registration and management
- Request blood and check availability
- Admin dashboard for managing inventory
- Real-time updates of blood stock

---

## 🛠️ Tech Stack
- **Backend:** Python, Django
- **Frontend:** HTML5, CSS3, Bootstrap, JavaScript
- **Database:** MySQL
- **Tools:** Git, VS Code
